# C on retro systems

C compilers for 8-bit and 16-bit targets let you write **structured** code while still talking to hardware when needed.

## Toolchains in the IDE

| Topic | Description |
|--------|-------------|
| **[cc65](cc65.md)** | 6502 C development — common for Commodore, Atari 8-bit, Apple II, … |
| **[z88dk](z88dk.md)** | Z80 C development — ZX Spectrum, MSX, CPC, and many more targets |
| **[CMOC](cmoc.md)** | 6809 C development — Dragon 32, TRS-80 CoCo 2, Vectrex |
